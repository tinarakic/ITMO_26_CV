# 🐝 HoneyBee Tracker
## Отслеживание положения пчел

Проект посвящен детекции и трекингу пчел на видео в реальном времени.  
В качестве основы используется датасет **Honeybee positions**, содержащий разметку положений и ориентации пчёл в кадрах.

## Описание

Цель проекта — разработать pipeline, который:
- обрабатывает видео с высокой плотностью объектов;
- выполняет детекцию пчёл;
- отслеживает их положение во времени;
- формирует итоговую визуализацию с траекториями и ID объектов;
- определяет итоговое направление "танца" пчёл.

## Зачем это нужно

Подобный анализ может использоваться для:
- мониторинга состояния улья по активности пчёл;
- изучения пространственных паттернов движения;
- анализа поведения пчёл и их ориентации внутри улья.

## Pipeline

Итоговое решение включает следующие этапы:

1. Загрузка данных.
2. Разбиение кадров на тайлы.
3. Детекция пчёл и оценка их положения.
4. Объединение результатов по тайлам.
5. Трекинг объектов.
6. Сохранение итогового видео и результатов.

## Структура решения

Основная логика реализована в ноутбуке:

- `final-pipeline.ipynb` — полный pipeline обработки видео.
- `honeybee_tracker_app.py` — основный интерфейс решения.

Остальные файлы содержать дополнителную информацию: предобработка данных, обучение и оценка качества модели. В папке `research` представлены промежуточые шаги в разработке итогового решения.

## Датасет

Используется датасет:

- [Honeybee positions](https://www.kaggle.com/datasets/kport354041/honeybee-positions)

Он содержит кадры с разметкой позиций и ориентации пчёл в плотных сценах. Датает состоит из 2 видеозаписи (30 fps и 70 fps), downsampled до 2 fps, по 360 кадров каждая (5120×5120), плюс текстовые файлы с координатами и ориентацией пчёл.

## Результат

На выходе система формирует:
- детекции пчёл;
- идентификаторы треков;
- визуализацию движения на видео;
- выводит итоговое доминирующее направление "танца" пчёл.
